# Casa-Hotel Reference Website

## 📄 Project Description

This website serves as a reference page for a lodging business (casa-hotel). Its main purpose is to showcase the company’s services and facilities, **without user reservation functionality**.

The system is divided into two main sections:

- **🔓 Public View:**  
  Accessible to all visitors, displaying information about the business, rooms, services, and a photo gallery.

- **🔐 Admin Panel:**  
  Restricted access for administrators and employees, allowing internal content management.

## 🛠️ Technologies Used

- **PHP (Native)** – for backend functionality and server-side logic  
- **Bootstrap** – for responsive layout and UI components  
- **CSS (Custom)** – for personalized styling

---

> ⚠️ Note: This project does not support online booking. It is designed solely as a company reference site.
